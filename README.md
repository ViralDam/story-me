# Story.Me
## Setup Backend
- Go to `story-me-backend` directory in command prompt.
- Run `pip install -r requirements.txt`
- Run `python app.py`

## Setup Client
- Go to `story-me-client` directory in command prompt.
- Run `npm install`
- Run `npm start`
